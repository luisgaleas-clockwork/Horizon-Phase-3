# horizon-beyond-V3
Third phase of ecommerce project "Horizon Beyond" using React.js, mySQL, Node.js, Express, Chia, and Mocha made during Road 2 Hire.


## Installation

Use npm install to get dependencies.

```bash
npm install
```
or

```bash
npm i
```

## Running server & React App

```bash
npm run dev
```
